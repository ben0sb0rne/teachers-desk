CSV Format Guide

Your CSV file must have three columns with these exact headers:

columnOne of B I N G O
problemThe problem text shown on screen (e.g. -4 + 3)
answerA numeric answer students mark on their card (e.g. -1)

Example

column,problem,answer
B,-10 + (-15),-25
B,-8 + (-12),-20
I,-7 + (-6),-13
N,-4 + 3,-1
G,4 + 5,9
O,20 + (-3),17

Each column needs at least 5 distinct answer values for bingo cards to work.
Answer ranges should not overlap between columns.
