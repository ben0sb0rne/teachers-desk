Math Bingo

How to use this tool.

What it does

Load a list of math problems, print bingo cards (one per student), and host a live calling round on the classroom projector. Each problem has a column letter (B, I, N, G, or O) and an answer; students mark the answer on their card when you call the problem.

Getting started

Open Math Bingo. From the homepage you can:

  • Pick a topic — browse the catalog of built-in problem sets (K through Calculus). Each topic card expands to show its variants.
  • Choose File — upload your own CSV of problems.
  • New Blank Set — open an empty editor and type problems in by hand.
  • Format Help — the CSV format guide.
  • Download Template — a starter CSV you can edit in any spreadsheet app.

Any set you upload or build is saved automatically to My Sets on the homepage. Each saved set has Host Game / Edit / Print / Rename / Delete actions; nothing leaves your browser.

CSV format

Three required columns, in this order:

column,problem,answer
B,-10 + (-15),-25
I,-7 + (-6),-13
N,\frac{2}{4},\frac{1}{2}
...

The column field is one of B, I, N, G, or O — that's the bingo column the problem belongs to. Each column needs at least 5 distinct answers to generate cards (15 per column is recommended for variety).

LaTeX problems and answers

Both the problem and answer fields accept LaTeX. Wrap math in standard LaTeX commands and the editor, the caller view, and the printed cards will render it as proper math notation. For example:

B,\frac{2}{4},\frac{1}{2}
G,\sqrt{16},4
O,3^2 + 4^2,25

Anything containing a backslash is treated as LaTeX. Plain numbers and arithmetic expressions (like -10 + (-15)) stay as regular text.

Printing cards

From the editor view, the left sidebar controls the print job:

  • Number of Cards — how many unique cards to generate.
  • Pre-Assign by Class — if you have a roster (from the Seating Chart tool), pick a class and each card prints with the student's name on it.
  • Card Style — Full Page (one large card, portrait), Half-Page (two per landscape page), or Work Area (one card next to a work-space area).
  • Card Colors — per-column color pickers, or switch to Black & White for a print-friendly version.
  • Advanced Options — card label, work-area style (lined / grid / blank), line spacing, caller-sheet companion.
  • Download Cards — generates a PDF and downloads it.

If your problems use LaTeX, generation takes a few seconds the first time as each unique math expression is rasterised for the PDF. Repeat downloads of the same set are quick.

Hosting a game

Click Host Game in the editor (or directly on a topic / My Sets card). The caller view fills the screen with a bingo ball and the current problem.

  • Space or click Next — call the next problem.
  • Back — step back through the call history.
  • Check Answers — opens the answer-verification panel. The big Bingo! button is the win-verify CTA when a student claims bingo.
  • ? — the full keyboard-shortcut reference.

The fullscreen button collapses the topstrip down to just the fullscreen toggle so the only chrome the class sees is the problem and the ball.

Settings

The gear icon opens the settings overlay:

  • Appearance & Sound — theme (auto / light / dark), display font, sound mute / volume.
  • Ball Style — Realistic (3D rolling photoreal ball) or Simple (flat colored chip with the column letter).
  • Called Numbers Board — show / hide the called-problems board, switch between Grid and Recent modes, set the number of recent balls shown.
  • Auto-Advance Timer — optional pacing aid that advances the call after a set interval.
