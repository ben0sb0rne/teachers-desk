Name Picker

How to use this tool.

What it does

Spin a wheel to pick a student at random from a class roster. Useful for cold-calls, fair turn-taking, group draws, or just adding a little ceremony to a question.

Getting started

Open the picker and either pick an existing class (your seating chart's classes show up here too) or click + New class to create one. Paste names one per line.

Spinning

Click Spin or press the spacebar. The wheel will turn for a few seconds and stop on a name. Each pick increments a call count in the sidebar so you can keep things fair.

Allow repeats / Reset session

If you want each student to be picked once before any name comes up again, switch off Allow repeats. Picked names get crossed out in the sidebar. Click Reset session to clear that list.
